Following functions are available:

/rainbow a rainbow animation on LEDs
/wave a slow wave animation on LED on base color specified by arguments: r=<0..255> g=<0..255> b=<0..255>
/setleds sets LEDs to the color from arguments: r=<0..255> g=<0..255> b=<0..255>
/ledsoff turns off LEDs
/setpins sets to any of the in arguments specified pins (D0..D8) to their PWM values (0..1023). To use them digital: 0=off, 1023=on
/togglepins inverts all pin values form pins used before.
